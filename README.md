# ERP Management System

A modern **ERP (Enterprise Resource Planning)** web application built with the **MERN Stack**, designed to streamline business operations. The system includes an **AI-powered business assistant** that helps users retrieve business information, answer ERP-related questions, and improve productivity through natural language interactions.

## ✨ Features

### Core ERP Modules

* 🔐 User Authentication & Authorization
* 📊 Dashboard with business insights
* 👥 Lead Management
* 🏢 Company Management
* 📦 Product Management
* 💰 Sales Management
* 📝 Activity Management (Calls, Emails, Meetings, Tasks & Notes)
* 🔍 Search and Filtering
* 📱 Responsive Design
* 🔒 Secure REST API with JWT Authentication

### 🤖 AI Business Assistant

* AI-powered ERP chatbot
* Answers questions about business data
* Provides product and sales information
* Retrieves lead and company details
* Understands natural language queries
* Assists users with ERP navigation
* Delivers quick insights to improve productivity

## 🛠️ Tech Stack

### Frontend

* React.js

* React Router
* Axios
* Context API

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
